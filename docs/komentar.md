# Komentar

## Baris Komentar
Baris komentar menggunakan `//`
```rust
fn main(){
    // ini adalah komentar
    // komentar tidak akan dieksekusi
    println!("hello world");
}
```

## Blok Komentar
Blok komentar dibuka dengan `/*` dan ditutup `*/`
```rust
/*
    Semua yang ada didalam
    blok komentar
    tidak akan dieksekusi
*/
fn main(){
    println!("hello world");
}
```
